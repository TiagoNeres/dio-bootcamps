# Estrutura de dados

É um objeto utilizado para armazenar `sequencialmente` dados do **mesmo tipo**. Permanecem com o **mesmo tamanho depois de criados**.

**Unidimensional** é quando usamos uma **única linha de dados**, como um **vetor**. Usamos **um índice** para acessar os elementos.

```java
int[] vetor = {10, 20, 30};
System.out.println(vetor[1]); // Saída: 20
```

**Multidimensional** é quando os dados são organizados em **linhas e colunas**, como em uma **matriz**. Usamos **dois índices**: um para a linha e outro para a coluna.

```java
int[][] matriz = {{1, 2},{3, 4}};
System.out.println(matriz[0][1]); // Saída: 2
```
